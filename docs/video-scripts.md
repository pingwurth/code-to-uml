```
- Open browser

- Use lightbox features

[2:30-3:00] AI Integration
"Now the magic happens when we integrate with AI..."
```

#### Key Features (3:30-4:30)

```
2. Browser-first rendering with WASM
3. Automatic server fallback
4. 20+ diagram types
5. Bilingual support (English/Chinese)
```

#### Conclusion (4:30-5:00)
```
"Code-To-UML is open source and free to use. Star it on GitHub, try it out, 
and let me know what you think. Link in the description."
```

## Recording Guidelines

### Screen Recording Setup

**Resolution:** 1920x1080 (16:9)
**Frame Rate:** 30fps or 60fps
**Audio:** Clear voiceover, no background noise

### Tools Recommended

- **Screen Recording:** OBS Studio, Camtasia, or ScreenFlow
- **Video Editing:** DaVinci Resolve (free), Adobe Premiere, Final Cut Pro
- **Thumbnail Creation:** Canva, Figma, or Photoshop

### Recording Checklist

- [ ] Clean desktop (hide unnecessary icons)
- [ ] Close unnecessary applications
- [ ] Use dark mode for better visibility
- [ ] Increase font size for readability
- [ ] Test audio levels before recording
- [ ] Have demo data ready
- [ ] Practice the flow beforehand

### Visual Style

**Colors:**
- Background: Dark (#1a1a2e) or Light (#f8f9fa)
- Accent: Blue (#1f6feb)
- Text: High contrast

**Typography:**
- Headings: Bold, 24-32px
- Body: Regular, 16-18px
- Code: Monospace, 14-16px

**Animations:**
- Smooth transitions (300ms)
- Subtle zoom effects
- Highlight important elements

## Publishing Checklist

### Before Publishing

- [ ] Video is properly edited
- [ ] Audio is clear and balanced
- [ ] Captions/subtitles added
- [ ] Thumbnail is eye-catching
- [ ] Title is compelling
- [ ] Description includes links
- [ ] Tags/keywords added
- [ ] End screen/cards added

### Platform-Specific Tips

#### YouTube
- Title: Include "Code-To-UML" and key benefit
- Description: Detailed with timestamps
- Tags: UML, PlantUML, AI, documentation, code analysis
- Thumbnail: Custom with logo and key visual
- End screen: Link to GitHub and related videos

#### Bilibili
- Title: Chinese title with key benefit
- Description: Detailed Chinese description
- Tags: UML, PlantUML, AI, 代码分析, 文档生成
- Cover: Eye-catching with Chinese text

#### Twitter
- Keep under 2:20 for native video
- Add captions (85% watch without sound)
- Thread format for longer content
- Pin the main tweet

#### LinkedIn
- Professional tone
- Focus on productivity benefits
- Tag relevant companies/people
- Use hashtags: #UML #AI #OpenSource #DeveloperTools
